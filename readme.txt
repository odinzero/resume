====================== PHP russian version =========================

1. перейти по ссылке https://odinzero.github.io/resume/

2. кликнуть на бегущую строку "Click here to decode this page"

3. выбрать язык программирования - PHP

4. выбрать язык - русский

5. ввести ключ - "Secret Passphrase"

6. нажать "Decrypt resume"
